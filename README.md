# Trabalhando com Links em HTML

Este repositório contém um exercício prático focado na estruturação de hiperlinks em HTML5. O objetivo principal foi consolidar o entendimento sobre como conectar diferentes páginas e seções dentro de um projeto web.

## O que foi praticado

Neste projeto, apliquei três conceitos fundamentais de navegação:
- **Links Relativos:** Navegação entre diferentes arquivos e pastas dentro do mesmo servidor (ex: caminhos para subpastas).
- **Âncoras Internas:** Criação de pontos de salto (IDs) para navegar entre diferentes seções de uma mesma página, melhorando a experiência em páginas extensas.
- **Links Externos:** Direcionamento para sites externos utilizando o atributo `target="_blank"` para garantir que o link abra em uma nova aba.

## Tecnologias
- **HTML5**

# Working with HTML Links

This repository contains a practical exercise focused on structuring hyperlinks in HTML5. The main goal was to consolidate the understanding of how to connect different pages and sections within a web project.

## Key Features

In this project, I implemented three fundamental navigation concepts:
- **Relative Links:** Navigating between different files and folders within the same server (e.g., paths to subfolders).
- **Internal Anchors:** Creating jump points (IDs) to navigate between different sections of the same page, improving the user experience on long pages.
- **External Links:** Linking to outside websites using the `target="_blank"` attribute to ensure the link opens in a new tab.

## Technologies
- **HTML5**
